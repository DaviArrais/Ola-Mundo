Aprendendo Git e GitHub

Aprendemos a preparar o ambiente de desenvolvimento no Ubuntu, instalando o google Chrome Stable, Visual Code Studio e o Github desenvolvido pelo Brendan Forster

Aprendemos os principais comandos das ferramentas:

  • COMMIT - Comando para criar versão local no git do projeto
  • PUSH - Comando para enviar commit para servidor
  • PULL - Comando para trazer do servidor commit para máquina local
  • FETCH ORIGIN - Comando para verificar diferenças entre o projeto local e a versão do servidor

Aprendemos também sobra marcações feitas no visual studio code

  • U - Houve auterações que o vercionador ainda não reconhece
  • M - Houve alterações em projeto que ele já reconheceu
